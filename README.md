`BASE_URL` 

`GOOGLE_API_KEY`
## Run Locally

Clone the project

```bash
git clone https://msruan/eivizinho-app
```

Go to the project directory

```bash
cd eivizinho-app
```

Install dependencies

```bash
flutter pub get
```

Run the app

```bash
flutter run
```
